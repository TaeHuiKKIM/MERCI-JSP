JSP로 개발한 반응형 쇼핑몰 웹사이트 프로젝트입니다. <br>
트렌디하게 슬라이드 되는 메인 페이지 외에도 아래 기능들이 포함되어 있습니다.

1. 물품 관리 - 등록/수정/삭제 - 품목 별 리뷰 
2. 고객 관리 - 회원가입 / 계정 삭제 / 로그인(소셜) 및 비밀번호 변경 - 찾기 / 상품 찜 기능 / 주소지 저장 및 불러오기 (도로명주소 API) 
3. 주문 관리 - 결제 확인 및 주문 완료 변경 / 리뷰 등록-수정-삭제
4. 주문 및 결제(장바구니 및 대행사 활용) - AJAX 활용하여 사용 편의성 향상
5. 문의 관리 - QNA 등록 수정 삭제
6. 물품 카테고리별 나열 및 검색
7. 소개 페이지 및 수정 페이지

이외에도 디테일적인 부분이 많고 결제는 대행사 테스트용 API를 활용했습니다. 

https://wellbeingexpress.com
